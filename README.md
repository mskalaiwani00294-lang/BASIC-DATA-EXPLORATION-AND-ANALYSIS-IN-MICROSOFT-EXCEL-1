# BASIC-DATA-EXPLORATION-AND-ANALYSIS-IN-MICROSOFT-EXCEL-1
# Project Title

BASIC-DATA-EXPLORATION-AND-ANALYSIS-IN-MICROSOFT-EXCEL-1

## Description

This project focuses on performing basic data exploration and analysis using Microsoft Excel. The dataset used in this project contains product sales and inventory information such as Product ID, Product Name, Brand Name, Price, Quantity, and Category. The purpose of the project is to analyze product distribution, pricing trends, inventory value, and category-wise performance using Excel formulas and functions. Various Excel functions including SUM, COUNTA, AVERAGE, MIN, MAX, IF, SUMIF, COUNTIF, LEFT, MID, and RIGHT were applied to generate meaningful business insights.

## Getting Started

### Dependencies

Before starting the project, ensure the following requirements are available: • Microsoft Excel 2016 or later • Windows 10/11 or macOS • Basic knowledge of Excel formulas and spreadsheet operations

### Installing
Steps to Set Up the Project
1.	Download or create the Excel dataset file.
2.	Open the dataset using Microsoft Excel.
3.	Ensure all columns are properly labeled: o Product ID o Product Name o Brand Name o Price o Quantity o Category
4.	Save the file in .xlsx format. No additional software installation is required


### Executing program

Step-by-Step Process
1.	Open the Excel workbook.
2.	Select the required data range.
3.	Apply Excel formulas for analysis.
4.	Review and interpret the results. Example Excel Formulas =SUM(D2:D35) =AVERAGE(D2:D35) =COUNTIF(D2:D35,"<100") =IF(D2>500,"HIGH PRICE","STANDARD") =SUMIF(F2:F35,"Electronics",D2:D35)


## Help
Problem: Formula not calculating • Ensure the cell format is not set to Text. • Press F2 and then Enter to refresh the formula.
